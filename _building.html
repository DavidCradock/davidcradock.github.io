Building

The build process

DC, Dave's Code is developed on the Microsoft Windows platform and uses Microsoft's awesome Visual Studio to build everything. This page describes how to build everything.

Install Visual Studio

Search for it on the web, download and install.

Third party libraries

Each of the required third party libraries are located within the /ThirdParty/ directory inside their own named directory.
We build each library indendently from the main DC project/solution as it's the simplest way to set stuff up, if needed.
Here are specific instructions for each library...

BassAudio

It was downloaded from https://www.un4seen.com/ and decompressed into the /ThirdParty/BassAudio/ directory.
The library was added to the project linker settings and the dll was copied to the root dir.

FastNoiseLite

It was downloaded from https://github.com/Auburn/FastNoiseLite
It's a lovely, simple header only library so it's header is added to the project and we're done.

stb_image, stb_image_write and stb_image_resize2

They were downloaded from... https://github.com/nothings/stb
Each library is a simple header and they are added to the project and we're done.

tinyobjloader

It was downloaded from https://github.com/tinyobjloader/tinyobjloader, decompressed into the /ThirdParty/tinyobjloader/ directory and added into DC project's ThirdParty filter.

Build process

From a clean project after cleanup.bat has been called...
Open the DC.sln found in the root directory into Microsoft's Visual studio and go to the menu -> Build -> Batch Build... Select all the projects and click build.
