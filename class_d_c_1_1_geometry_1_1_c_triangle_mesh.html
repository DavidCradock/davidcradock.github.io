Detailed Description

A triangle mesh consisting of vertices and indices.

This class is used to store a collection of vertices and indices which represent a 3D mesh of triangles. It provides methods to manipulate the mesh such as applying transformations, computing normals, tangents and binormals. Aswell as adding geometric shapes such as spheres, cubes, cylinders etc. It can also load and save mesh data from/to files. Each vertex added is supposed to be unique, with vertex indices pointing to the unique vertex data, with three indices per triangle. Multiple indices can point to the same vertex data.

NOTE: Not that we're using OpenGL, but for consistancy, glFrontFace() is set to GL_CCW by default which is the convention which we will also use. Therefore, the winding order of the vertices should be counter-clockwise for them to be considered front facing.

void DC::Geometry::CTriangleMesh::decimate ( void )

Performs decimation on the mesh.

Reduces the mesh by removing every other vertex and re-triangulating the remaining vertices. This provides a basic decimation method that reduces the number of vertices and indices by half. For more advanced decimation techniques, I should explore edge collapse algorithms or other mesh simplification methods.
