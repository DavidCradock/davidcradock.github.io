Detailed Description

template<typename T>
class DC::Core::CManager< T >

Template class to manage pointers to objects of a class of the given type.

An object has a reference count, so if it is added multiple times, it will not be removed with calls to remove() until the reference count has reached 0. That is unless removeAll() is passed true to it's bForceRemoval parameter in which case, it is. Example usuage...

#include "TemplateManager.h"
X::CManager<CCamera> cameraManager;
CCamera* pCamera = cameraManager.add("Camera1");
