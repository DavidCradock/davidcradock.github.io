Interpolates between this colour and the one given by the given float and returns the new colour.

Parameters
otherThe other colour to interpolate between this one.
fValueThe interpolation value between this and the other colour.
Returns
The resulting colours interpolated between.

If the float is 0.0f, the returned colour will be this colour. If the float is 0.5f, the returned colour will be half way between this colour and the one given. If the float is 1.0f, the returned colour will be the one given.
